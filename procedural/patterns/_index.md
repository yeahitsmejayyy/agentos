# patterns — Conventions with detail

Standards that outgrew a one-line principle: a convention spelled out, maybe with a snippet — but not
yet code. The middle rung of the fidelity ladder. **`memory_type: procedural`**, `kind: pattern`,
`materializable: false` (recall only — a pattern is read, not installed).

A pattern that hardens into reusable code **graduates** to `modules/` — its prose becomes the module README.

Copy `_template/pattern.md` to start one. Ships empty — patterns are mined from real work.
